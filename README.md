# hfut-course-helper
![Version](https://img.shields.io/badge/Version-7.0-brightgreen.svg)
![Tampermonkey](https://img.shields.io/badge/Platform-Tampermonkey-blue.svg)
![License](https://img.shields.io/badge/License-MIT-orange.svg)

**全自动抢课助手** 是一款专为高校教务系统（EAMS5架构）设计的油猴（Tampermonkey）脚本。它内置了现代化的图形界面（GUI），支持**关键词搜索课程自动填入**、**点击标签一键添加**、**可视化监控**等功能，彻底告别繁琐的 F12 开发者工具，让选课变得简单高效。

> **⚠️ 注意**：本脚本主要针对合肥工业大学（HFUT）及其他使用同类 EAMS5 教务系统的学校优化。使用前请确保您所在的网络环境畅通。

---

## ✨ 核心功能 (v7.0)

* **🔍 智能搜索 (新!)**：无需在列表中翻找，直接在脚本面板输入课程名称（如“羽毛球”），自动从全校课表中查找并填入 ID。
* **🖱️ 点击即填**：开启“显示页面ID”后，网页上每个选课按钮旁会出现红色 ID 标签，点击即可加入抢课列表，无需复制粘贴。
* **🖥️ 现代化 GUI**：
    * **可拖拽**：悬浮窗挡住视线？按住标题栏随意拖动。
    * **可拉伸**：想看更多日志？拖动右下角随意改变窗口大小。
    * **自动记忆**：刷新页面后，你的窗口位置、大小、已填 ID 和时间设置都会自动恢复。
* **🤖 拟人化操作**：采用 `基础间隔` + `随机抖动` 机制，模拟真人点击频率，降低被服务器判定的风险。
* **📜 可视化日志**：实时显示点击次数、系统返回消息（成功/满员/冲突），关键信息高亮显示。

### 第一步：安装脚本管理器
你需要先安装浏览器扩展程序 **Tampermonkey (篡改猴)**：

* **方法 A (推荐)**：前往 [Greasy Fork](https://greasyfork.org/zh-CN) 搜索 "全自动抢课助手" 点击安装。
* **方法 B (手动)**：下载本项目中的 `hfut-course-helper.user.js` 文件，拖入浏览器安装。

## 📖 使用教程

### 1. 启动脚本
登录教务系统，进入**“学生选课”**页面。脚本会自动加载，屏幕右上角（或上次保存的位置）会出现黑色的控制面板。

### 2. 添加课程 ID (三种方式)
* **方式一：智能搜索（推荐）**
    在面板的搜索框输入课程关键词（例如“芯片”），点击**【查找并填入】**。脚本会自动扫描当前页面表格，将匹配的课程 ID 填入列表。
* **方式二：点击标签**
    点击面板上的小按钮**【显示页面ID标签】**，网页上所有“选课”按钮旁会出现红色标签 `➕123456`。点击红色标签，ID 自动飞入输入框。
* **方式三：手动输入**
    如果你已经知道课号，直接在输入框中填写，多个 ID 用逗号分隔（例如：`487758, 487759`）。

### 3. 设置频率
* **基础间隔 (ms)**：建议设置 `1000` 或 `1500`（即 1~1.5秒）。这是最快点击速度，太快可能会被服务器踢出。
* **随机抖动 (ms)**：建议设置 `500` 或 `1000`。这会让每次点击间隔在 `基础` 到 `基础+抖动` 之间随机变化，模拟真人操作。

### 4. 开始/停止
* 点击绿色 **【启动抢课】** 按钮，脚本开始工作，请保持网页在前台运行。
* 观察下方日志区域，如果看到 `🎉 选课成功`，点击红色 **【停止】** 按钮。

## ❓ 常见问题 (Q&A)

**Q: 为什么点击搜索后提示找不到课程？**
A: 脚本只能搜索**当前已加载**的表格数据。请确保你已经切换到了对应的标签页（如“全校选课”或“培养方案选课”），并且页面列表已经显示出来了。

**Q: 脚本运行一会后页面卡死或白屏？**
A: 这通常是教务系统服务器拥堵导致的。请尝试刷新页面，脚本会自动恢复上次的 ID 设置，直接再次点击启动即可。

**Q: 会封号吗？**
A: 脚本采用了随机延迟策略来模拟人类操作，但**任何自动化工具都有风险**。建议不要将频率设置得过高（低于 800ms），且仅在抢课高峰期使用。
